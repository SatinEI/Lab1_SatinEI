Проект на C# с графическим интерфейсом WPF, который определяет тип треугольника по трём заданным сторонам и вычисляет координаты его вершин, масштабированные для отображения в поле 100×100 пикселей. Реализовано логирование всех запросов в консоль и текстовый файл.
## Функциональность

- **Ввод трёх сторон** – числа с плавающей точкой (float) или произвольные строки.
- **Проверка данных** – определение, являются ли введённые строки положительными числами.
- **Определение типа треугольника**:
  - `равносторонний`
  - `равнобедренный`
  - `разносторонний`
  - `не треугольник` (если стороны не образуют треугольник)
  - пустая строка (при нечисловых входных данных)
- **Вычисление координат вершин** – три точки `(X,Y)` в целых числах, масштабированные под поле 100×100 пикселей.
  - Для невалидных данных возвращается `(-2,-2)` для всех вершин.
  - Для числовых данных, не образующих треугольник, – `(-1,-1)`.
- **Графическая отрисовка** – треугольник рисуется на встроенном Canvas (WPF Polygon) при корректных координатах.
- **Логирование** – каждая попытка вычисления записывается в консоль и в файл `triangle_log.txt`
